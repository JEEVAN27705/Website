# 🌐 Jeevan Patil - Portfolio Website

Personal portfolio website showcasing my **projects, skills, and experience**

## 🚀 Overview
A modern and interactive portfolio to highlight my work in **web development, UI design, and technical projects**, along with contact and collaboration opportunities.


## ✨ Features
- 🧑‍💻 **About Me Section**  
- 📂 **Projects Showcase** with details  
- 📞 **Contact Section** with email & phone  
- 🎨 **Modern UI with animations**  
- 📱 **Fully Responsive (Mobile + Desktop)**  
- ⚡ **Smooth transitions using Framer Motion**


## 🛠️ Tech Stack

- ⚛️ React  
- 🎨 Tailwind CSS  
- 🎞️ Framer Motion  
- 🧩 Lucide Icons  


## 📂 Getting Started

1. Clone the Repository
```sh
git clone https://github.com/YOUR_USERNAME/portfolio.git
```

2. Navigate to Project Folder

```sh
cd portfolio
```

3. Install Dependencies
```sh
npm install
```

4. Run Locally
```sh
npm run dev
```
